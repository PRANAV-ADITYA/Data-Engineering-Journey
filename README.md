🚀 Data Engineering Journey — From  Foundations to Cloud & AI

📖 About

This repository documents my journey towards becoming a Data Engineer with a strong foundation in Cloud and AI.
Rather than simply completing courses, the goal of this repository is to deeply understand the concepts behind modern data platforms by writing code, experimenting with technologies, solving problems, and documenting the learning process along the way.
Every notebook, piece of code, and commit in this repository represents a step forward in that journey.

🎯 What You'll Find Here

This repository contains my learning notes, practice notebooks, experiments, and code while exploring topics such as:

- 🐍 Python
- 🗄️ SQL
- ⚡ Apache Spark & PySpark
- 🔄 Apache Airflow
- 📡 Apache Kafka
- ☁️ Cloud Platforms (Azure)
- ❄️ Snowflake
- 🏗️ Data Warehousing
- 📊 ETL & Data Pipelines
- 🧠 Engineering concepts that support modern AI systems

As the journey progresses, new topics and technologies will continue to be added.

Each directory contains topic-wise notebooks and learning material.
Larger end-to-end projects are maintained in their own dedicated repositories with detailed documentation.

🌱 Why This Repository Exists

I wanted a single place to document my learning journey—not only to track my own growth, but also to create a knowledge base that I can revisit whenever needed.
Over time, I hope this repository evolves into a comprehensive collection of concepts, code snippets, experiments, and notes covering the core pillars of Data Engineering.
